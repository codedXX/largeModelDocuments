# Java入门：程序流程控制

本篇文档为您梳理了 Java 程序流程控制的核心语法，包括：**分支结构、循环结构、控制跳转语句**以及**随机数生成（Random）**的知识点。

---

## 1. 程序的执行顺序
程序的执行顺序可以分为三种基本结构：
1. **顺序结构**：代码自上而下按顺序依次执行。
2. **分支结构**：根据条件，选择对应的代码执行（如 `if`、`switch`）。
3. **循环结构**：控制某段代码重复执行多次（如 `for`、`while`、`do-while`）。

---

## 2. 分支结构

### 2.1 `if` 分支结构
**应用场景**：根据条件的真或假来决定执行哪段代码，非常**适合做区间判断**。

**三种格式与执行流程**：
1. **单分支**：
   ```java
   if (条件表达式) {
       语句体;
   }
   ```
2. **双分支**：
   ```java
   if (条件表达式) {
       语句体1; // 条件为 true 执行
   } else {
       语句体2; // 条件为 false 执行
   }
   ```
3. **多分支**：
   ```java
   if (条件表达式1) {
       语句体1;
   } else if (条件表达式2) {
       语句体2;
   } else {
       语句体n; // 所有条件都不满足时执行
   }
   ```

### 2.2 `switch` 分支结构
**应用场景**：通过**比较值是否相等**，来决定执行哪条分支。它格式良好，代码优雅，性能较好，适合用于固定值的比较。

**格式**：
```java
switch (表达式) {
    case 值1:
        执行代码1;
        break;
    case 值2:
        执行代码2;
        break;
    default:
        执行代码n; // 都不匹配时执行
}
```

**使用 `switch` 的注意事项（重点）**：
1. **支持的类型**：表达式类型只能是 `byte`、`short`、`int`、`char`，JDK 5 开始支持枚举（Enum），JDK 7 开始支持 `String`。**不支持 `double`、`float`、`long`**。
2. **case 的值**：`case` 给出的值不允许重复，且只能是**字面量**，不能是变量。
3. **穿透现象**：正常使用 `switch` 时，不要忘记写 `break`，否则会出现代码“穿透”现象。
   - **穿透性的应用**：如果多个 `case` 分支的代码逻辑相同，可以去掉前面的 `break`，将相同的代码集中写在一个 `case` 块中，利用穿透性来简化代码。

> 注意：
>
> * double、float 不支持，是因为小数有精度问题，不适合精确匹配。
>   * System.out.println(0.1 + 0.2);//答案不等于0.3，通常为0.30000000000000004
>   * 因为在 Java 里，double 使用二进制存储小数。很多十进制小数，比如 0.1、0.2，转换成二进制后是**无限循环小数**，没法精确保存，只能保存一个近似值
> * long 不支持，不是因为精度，而是 Java 传统 switch 的设计限制，主要围绕 int 类型处理。

## 3. 循环结构

### 3.1 `for` 循环
**应用场景**：如果一开始**已知循环次数**，建议使用 `for` 循环解决。
```java
for (初始化语句; 循环条件; 迭代语句) {
    循环体语句(重复执行的代码);
}
```
*示例：* `for (int i = 0; i < 3; i++) { System.out.println("Hello World"); }`

### 3.2 `while` 循环
**应用场景**：如果一开始**不知道循环次数**，建议使用 `while` 循环解决。

```java
初始化语句;
while (循环条件) {
    循环体语句(被重复执行的代码);
    迭代语句;
}
```
*注：`for` 能解决的 `while` 也能解决，功能上无本质区别。区别在于 `for` 循环中控制循环的变量在循环结束后不可访问，而 `while` 循环外声明的变量循环后仍可使用。*

### 3.3 `do-while` 循环
**特点**：**先执行，后判断**。无论条件是否满足，循环体至少执行一次。
```java
初始化语句;
do {
    循环体语句;
    迭代语句;
} while (循环条件);
```

### 3.4 死循环与循环嵌套
- **死循环**：没有干预就不会停下来的循环。
  - 经典写法：`while(true) {}` 或 `for(;;) {}`
- **循环嵌套**：循环中又包含循环。
  - **特点**：外部循环每执行一次，内部循环会全部执行完一轮。

## 4. `break` 与 `continue`

| 关键字 | 作用范围与功能 |
| :---: | :--- |
| **`break`** | break 会**结束它所在的最内层 for / while / do-while / switch**。b**reak 是彻底跳出它所在的最近一层循环**。 |
| **`continue`** | 用于跳出当前循环的**当次执行**，直接进入循环的**下一次**执行。只能在循环中使用。 |

注意：

* 这里的 break 只会跳出**内层** for。

  * ~~~java
    for (int i = 0; i < 3; i++) {
        for (int j = 0; j < 3; j++) {
            break;
        }
    }
    ~~~

## 5. Random 生成随机数

在需要生成随机数（如猜数字游戏、验证码）时，可以使用 `Random` API，步骤如下：

1. **导包**：告诉程序去寻找该工具类。
   `import java.util.Random;`
2. **创建对象**：拿到随机数对象。
   `Random r = new Random();`
3. **生成数据**：调用方法获取随机数。
   `int number = r.nextInt(10);` // 生成 0 - 9 之间的随机数

**技巧：如何生成任意区间的随机数？**
使用**“减加法”**技巧。例如，如何生成 65 到 91 之间的随机数？
- 计算区间范围：`91 - 65 + 1 = 27`，我们需要产生 `0-26` 的随机数，再整体加上 `65`。
- 代码实现：`int number = r.nextInt(27) + 65;`
