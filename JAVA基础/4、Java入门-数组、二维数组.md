# Java入门：数组与二维数组

本篇文档为您梳理了 Java 中有关**数据存储容器**的核心知识，包括：**一维数组、二维数组**的定义、访问、遍历以及常见的相关算法逻辑（如求最值、元素交换）。知识点已经过结构化梳理，重点内容已做标注。

---

## 1. 数组（Array）

### 1.1 为什么需要数组？
当程序中需要处理**大批量、类型相同**的数据时，使用数组进行集中存储和管理，明显优于定义成百上千个独立的变量。

### 1.2 什么是数组？
**数组**是一个数据容器，用来存储一批**同类型**的数据。

### 1.3 数组的定义方式一：静态初始化
**适用场景**：在定义数组时，**已经确定了要存储的具体数据**。

- **简化格式**（常用）：
  ```java
  数据类型[] 数组名 = { 元素1, 元素2, 元素3, ... }; 
  int[] arr = {12, 24, 36};
  ```
- **完整格式**：
  ```java
  数据类型[] 数组名 = new 数据类型[]{ 元素1, 元素2, 元素3, ... };
  int[] arr = new int[]{12, 24, 36};
  ```
  *(注：`数据类型[] 数组名` 也可写成 `数据类型 数组名[]`，但在 Java 中推荐前者)*

### 1.4 数组的定义方式二：动态初始化
**适用场景**：定义数组时**只确定要存储的数据类型和数组的长度**，具体的数据后续再存入。

- **格式**：
  ```java
  数据类型[] 数组名 = new 数据类型[长度];
  int[] arr = new int[3];
  ```
- **动态初始化数组元素的默认值规则（重点）**：
  | 数据类型分类 | 具体数据类型 | 默认值 |
  | :--- | :--- | :--- |
  | **基本类型** | `byte`、`short`、`int`、`long` | `0` |
  | **基本类型** | `float`、`double` | `0.0` |
  | **基本类型** | `char` | `\u0000` (空字符) |
  | **基本类型** | `boolean` | `false` |
  | **引用类型** | 类、接口、数组、`String` | `null` |

>动态初始化数组：只确定数组的类型和存储数据的容量，不事先存入具体的数据。

### 1.5 数组的访问与操作

1. **获取数组的长度**（即元素的个数）：
   ```java
   System.out.println(arr.length);
   ```
2. **访问/修改数组元素**：
   - 数组的元素编号称为**索引**，索引从 **`0`** 开始计数。
   - 访问格式：`数组名[索引]`
   ```java
   System.out.println(arr[0]); // 获取第一个元素
   arr[1] = 100;               // 修改第二个元素的值
   ```

### 1.6 数组的遍历
**遍历**是指一个一个地访问数组中的所有数据。通常结合 `for` 循环来实现。
```java
int[] arr = {20, 30, 40, 50};
for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}
```

### 1.7 核心算法逻辑：元素交换
如果在数组中需要交换两个元素的位置，必须借助一个**临时变量**。
```java
int temp = arr[i];
arr[i] = arr[j];
arr[j] = temp;
```

## 2. 二维数组（2D Array）

### 2.1 什么是二维数组？
**二维数组**是指数组中的每一个元素又是一个一维数组。它可以用来表示表格数据、矩阵数据或是类似于“座位表”的空间结构。

### 2.2 二维数组的定义方式

- **静态初始化**：
  ```java
  数据类型[][] 数组名 = new 数据类型[][]{
      {元素1, 元素2},
      {元素3, 元素4}
  };
  // 简写：
  int[][] arr = { {12, 24}, {666, 888}, {10, 20} };
  ```
- **动态初始化**：
  ```java
  数据类型[][] 数组名 = new 数据类型[行长度][列长度];
  int[][] arr = new int[3][5]; // 包含3个一维数组，每个一维数组长度为5
  ```

### 2.3 二维数组的访问
- **访问某一行的数组对象**：`数组名[行索引]` (例如：`arr[2]`)
- **访问具体的某个元素**：`数组名[行索引][列索引]`
  ```java
  System.out.println(arr[2][1]); // 访问第3行第2列的元素
  arr[3][0] = 111;               // 修改元素
  ```

### 2.4 二维数组的遍历
二维数组的遍历需要使用**双重嵌套循环**：外层循环遍历每一行（一维数组），内层循环遍历一维数组中的每一个元素。
```java
for (int i = 0; i < arr.length; i++) {
    for (int j = 0; j < arr[i].length; j++) {
        int data = arr[i][j];
        System.out.print(data + "\t"); // 不换行输出，并加制表符
    }
    System.out.println(); // 每遍历完一行后换行
}
```

## 3. 综合案例（练习方向）
- **计算成绩信息**：录入多名学生成绩，利用数组遍历求出总分、最高分和最低分（打擂台思想）。
- **斗地主洗牌**：将 54 张扑克牌存入数组，结合随机数生成的索引，对数组元素进行交换打乱。
- **石头迷阵**：利用二维数组初始化游戏地图，并结合随机数打乱二维数组中的元素。
